# Setup

### install package
```
npm install -g npm # if your npm version < 3
npm install -g rnpm
npm install
```
# Start
```
npm start
```

#### Run ios
```
react-native run-ios
```

#### Run android
```
npm run bundle-android
npm run android-mock-update-server
react-native run-android
```
# Bundle
```
npm run bundle-ios
npm run bundle-android
```
